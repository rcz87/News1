## 🎯 Cara Membuat Artikel di Mobile

### 1. Login & Pilih Channel
```bash
1. Buka admin.html
2. Login dengan admin/admin123
3. Pilih channel (Ambal, Berita Angin, dll)
4. Klik "📋 Tampilkan Artikel"
```

### 2. Buat Artikel Baru
```bash
1. Klik "➕ Buat Artikel Baru"
2. Isi form:
   - Slug: judul-artikel (huruf kecil, pakai hubung)
   - Judul: Judul Artikel (Title Case)
   - Ringkasan: 1-2 kalimat
   - Isi: Artikel lengkap (bisa pakai Markdown)
   - Kategori: Pilih kategori
   - Penulis: Nama penulis
3. Upload foto landscape
4. Klik "💾 Simpan Artikel"
```

### 3. Edit Artikel
```bash
1. Klik "✏️ Edit" pada artikel
2. Ubah konten yang diperlukan
3. Klik "💾 Simpan Artikel"
```

## 📋 Format Penulisan yang Benar

### ✅ Judul & Slug
- **Judul**: `Puskesmas Ambal Kini Buka 24 Jam`
- **Slug**: `puskesmas-ambal-layanan-24-jam`

### ✅ Isi Artikel
- **Paragraf pendek**: 3-4 kalimat per paragraf
- **Kalimat aktif**: "Budi mengatakan..." bukan "Dikatakan oleh Budi..."
- **Huruf besar**: Hanya untuk nama orang, tempat, instansi
- **Singkatan**: UMKM, SMA, PKK (huruf besar semua)

### ✅ Kategori
- **Politik**: Berita pemerintahan, kebijakan
- **Ekonomi**: Bisnis, UMKM, pasar
- **Olahraga**: Pertandingan, atlet
- **Teknologi**: Digital, gadget, inovasi
- **Lifestyle**: Kuliner, wisata, hiburan
- **Berita**: Umum/lain-lain

## 🔧 Troubleshooting Mobile

### ❌ Common Issues & Solutions

#### 1. "Tidak bisa upload foto"
- **Cause**: File terlalu besar (>20MB) atau bukan gambar
- **Solution**: Compress foto dulu atau pilih foto lain

#### 2. "Form terlalu kecil di HP"
- **Cause**: Browser zoom issue
- **Solution**: Refresh page, pastikan viewport correct

#### 3. "Tombol tidak responsif"
- **Cause**: Touch issue
- **Solution**: Tap dengan jari, bukan nail, pastikan koneksi stabil

#### 4. "Login gagal"
- **Cause**: Username/password salah atau token expired
- **Solution**: Gunakan admin/admin123, clear browser cache
